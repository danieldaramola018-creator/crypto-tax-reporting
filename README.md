# Crypto Tax Reporting

Automated cryptocurrency tax calculation system built on Stacks blockchain using Clarity smart contracts.

## Features

• **Transaction Recording**: Track buy/sell transactions with exchange integration support
• **Capital Gains Calculation**: Automated calculation of gains and losses for tax reporting
• **Multiple Cost Basis Methods**: Support for FIFO, LIFO, and Average Cost methods
• **Asset Holdings Management**: Real-time tracking of cryptocurrency portfolio balances
